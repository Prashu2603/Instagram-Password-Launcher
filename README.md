# Instagram Password Launcher

A desktop application built with Python and Tkinter that prompts the user for a password before launching the Instagram Microsoft Store application.

## Features

- Password protected launcher
- Tkinter GUI
- Launches Instagram after successful authentication
- Standalone EXE using PyInstaller

## Technologies

- Python
- Tkinter
- PyInstaller

## Limitation

This application protects the launcher only. The original Microsoft Store Instagram application can still be opened directly from the Start Menu because Windows does not provide a built-in password protection feature for Store apps.
